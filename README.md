# **Nexium**

**Nexium is an AI-powered Roblox Studio assistant built to help developers create, improve, and debug their games faster.**

With Nexium, creators can generate Luau scripts, analyze existing code, fix errors, optimize systems, and get development assistance directly inside Roblox Studio.

Nexium is designed to make Roblox development easier by combining powerful AI tools with a clean, developer-focused experience.

---

## ✨ Features

## 🤖 AI Development Assistant

Nexium provides an intelligent AI assistant that understands Roblox development and Luau.

You can use it to:

- Generate scripts from simple descriptions
- Explain complex code
- Debug errors
- Improve existing systems
- Get ideas for game mechanics
- Learn Roblox development faster

Example:

```
Create a round system with an intermission timer and winner detection
```

Nexium will generate the required Luau code and explain how it works.

---

## 📝 Luau Script Generator

Create Roblox scripts quickly using natural language.

Nexium can generate:

- Server Scripts
- LocalScripts
- ModuleScripts
- UI systems
- Game mechanics
- Complete script structures

Generated code can be copied or inserted directly into Roblox Studio.

---

## 🔍 Script Analyzer

Analyze your existing Roblox scripts with AI.

Nexium can:

- Explain what a script does
- Find potential bugs
- Suggest improvements
- Optimize performance
- Rewrite outdated or messy code

## 🔌 AI Provider Support

Nexium supports multiple AI providers, allowing developers to use the models they prefer.

Supported providers:

- OpenAI
- Anthropic Claude
- Google Gemini
- OpenRouter
- Custom AI Providers

Custom providers allow developers to connect their own AI endpoints and models.

---

## ⚙️ Provider System

Nexium uses a modular AI architecture that makes it easy to add new providers.

Each provider supports:

- API key configuration
- Model selection
- Connection testing
- Custom endpoints
- Flexible settings
